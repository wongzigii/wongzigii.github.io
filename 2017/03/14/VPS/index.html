SSH Key

首先，先本地生成一个新的 SSH key 用来连接到远程服务器，然后加到服务器上 .ssh/authorized_keys 文件中。

参数：

  • -t: 加密算法：RSA 、RSA1、DSA、ECDSA、ED25519
  • -b: 多少位：4096
  • -C: 注释

更安全的做法：

  • 定期更换 SSH 端口
  • 禁用密码登录，本地配置好 ssh_config 文件
  • 特定 IP 加白名单
  • 用 fail2ban 来 ban 掉一些尝试登录的爬虫
